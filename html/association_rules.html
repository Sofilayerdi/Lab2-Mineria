2. Reglas de Asociación¶

Se aplica el algoritmo Apriori para descubrir patrones no evidentes en las películas del dataset de CineVision Studios. Las variables numéricas se discretizan en categorías significativas antes de aplicar el algoritmo.

Variables utilizadas:

  • budget → budget_zero / budget_low / budget_medium / budget_high
  • revenue → revenue_zero / revenue_low / revenue_medium / revenue_high
  • voteAvg → voteAvg_zero / voteAvg_low / voteAvg_medium / voteAvg_high
  • popularity → popularity_low / popularity_medium / popularity_high (terciles)
  • runtime → runtime_short (<80min) / runtime_medium (80-120min) / runtime_long (>120min)
  • releaseYear → era_classic (<2000) / era_2000s / era_2010s / era_recent (>=2020)
  • originalLanguage → lang_en / lang_es / lang_fr / lang_ja / lang_other
  • genres → cada género como ítem binario (genre_Drama, genre_Action, etc.)
  • video → has_video

Variables excluidas: id, title, originalTitle, director, actors, homePage, productionCompany, actorsCharacter — son identificadores únicos que no generan patrones generalizables.

Implementación del algoritmo Apriori¶

In [5]:
def apriori(transactions, min_support=0.05, min_confidence=0.5, max_len=3):
    """
    Implementación del algoritmo Apriori.
    Parámetros:
        min_support: fracción mínima de transacciones que deben contener el itemset
        min_confidence: confianza mínima para generar una regla A → B
        max_len: tamaño máximo de los itemsets a explorar
    """
    n = len(transactions)
    
    # Paso 1: contar ítems individuales
    item_counts = defaultdict(int)
    for t in transactions:
        for item in t: item_counts[frozenset([item])] += 1
    
    freq_sets = {k: v/n for k,v in item_counts.items() if v/n >= min_support}
    all_freq = dict(freq_sets)
    current_sets = list(freq_sets.keys())
    k = 2
    
    # Paso 2: generar itemsets de tamaño k
    while current_sets and k <= max_len:
        candidates = defaultdict(int)
        items_list = sorted(set(i for s in current_sets for i in s))
        for combo in combinations(items_list, k):
            candidate = frozenset(combo)
            if all(frozenset(sub) in all_freq for sub in combinations(combo, k-1)):
                for t in transactions:
                    if candidate.issubset(t): candidates[candidate] += 1
        new_freq = {ks: cnt/n for ks,cnt in candidates.items() if cnt/n >= min_support}
        all_freq.update(new_freq)
        current_sets = list(new_freq.keys())
        k += 1
    
    # Paso 3: generar reglas
    rules = []
    for itemset, support in all_freq.items():
        if len(itemset) < 2: continue
        for i in range(1, len(itemset)):
            for ant in combinations(itemset, i):
                ant = frozenset(ant)
                con = itemset - ant
                ant_sup = all_freq.get(ant, 0)
                if ant_sup == 0: continue
                conf = support / ant_sup
                if conf >= min_confidence:
                    lift = conf / all_freq.get(con, support)
                    rules.append({
                        'antecedents': ant, 'consequents': con,
                        'support': round(support,4),
                        'confidence': round(conf,4),
                        'lift': round(lift,4)
                    })
    
    if not rules: return pd.DataFrame()
    return pd.DataFrame(rules).sort_values('lift', ascending=False)

Conclusiones de Reglas de Asociación¶

Se aplicó el algoritmo Apriori con tres configuraciones distintas de parámetros:

Experimento 1 (support≥0.10, confidence≥0.70): Se generaron 204 reglas con alta confianza pero dominadas por ítems muy frecuentes, resultando en reglas triviales como budget_zero → revenue_zero.

Experimento 2 (support≥0.05, confidence≥0.60): 531 reglas. Mayor diversidad de patrones pero aún contaminadas por los ítems omnipresentes.

Experimento 3 (sin ítems >50%, support≥0.05, confidence≥0.55): 143 reglas accionables. Las más relevantes para CineVision Studios:

  1. Inversión y retorno (lift ~4.2–4.5): budget_medium + runtime_medium → revenue_medium y revenue_medium → budget_medium. Películas con presupuesto medio (5–50M) y duración estándar tienen alta probabilidad de generar ingresos medios. Esto confirma que la inversión moderada con duración apropiada es un perfil de bajo riesgo.

  2. Blockbusters (lift ~2.6–2.8): budget_high → popularity_high (conf=0.89) y revenue_high → popularity_high (conf=0.94). Las películas con mayor inversión casi siempre resultan en alta popularidad en TMDB, lo cual se correlaciona con mayor exposición y marketing.

  3. Películas de acción recientes (lift ~2.4): era_2010s + genre_Action → popularity_high. Las películas de acción de la última década tienen alta probabilidad de ser populares, sugiriendo que este género sigue siendo relevante para el mercado actual.

  4. Patrón de baja calidad (lift ~2.2–2.4): runtime_short + voteAvg_zero → popularity_low. Películas cortas sin calificaciones tienden a baja popularidad, representando el segmento de producciones sin distribución significativa.

Recomendación para CineVision Studios: Priorizar producciones de presupuesto medio-alto (5–50M USD) en géneros de Acción con duraciones de 80–120 minutos para maximizar probabilidad de buenos ingresos y alta popularidad.
